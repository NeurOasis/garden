How to Use These Visual Schedule Icons

This pack contains 19 visual icons designed for Zach's visual schedule:

  • Go for a Walk (3 variants: individual, father-son, father-son-dog)
  • Go for a Hike (3 variants: individual, father-son, father-son-dog)

  • Go to a Park
  • Go Sailing
  • Go to the Farmers Market
  • Ride Scooters
  • Ride Bikes
  • Have a Picnic
  • Go to a Nature Center
  • Go for a Run
  • Go for a Swim

Instructions:

  1. Print all pages on card stock or heavy paper if possible
  2. Cut along the dashed lines using the corner marks as guides
  3. Laminate the icons (optional) for durability
  4. Add to your visual schedule by placing them in the "Activity" or "Options" column as needed

Tips for Using the Schedule:

  • Use the "Options" column to provide choices within a structured framework
  • For activities with multiple versions (walk, hike, catch), you can let Zach choose which version he prefers
  • Consider using VelcroÂ® or magnets to make the schedule interactive

  • Set up the schedule together at the start of the day
  • Include a mix of preferred and non-preferred activities
  • Celebrate successful transitions and completing activities on the schedule

These visual supports are designed to provide structure, predictability, and choice within a framework that works for Zach.
